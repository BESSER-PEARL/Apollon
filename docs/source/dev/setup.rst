#############
Setup
#############

Clone the repository and change into the Apollon directory:

.. code-block:: shell

    git clone https://github.com/ls1intum/Apollon.git
    cd Apollon

Install all node dependencies

.. code-block:: shell

    yarn install

Launch the webpack-dev-server:

.. code-block:: shell

    yarn start

Run the tests:

.. code-block:: shell

    yarn test
